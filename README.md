# PowerBI Logistics Dashboard

## Objective
Global freight operations analysis dashboard built in Power BI using shipment data from 2023–2025. This project analyzes transit time, delivery performance, cost efficiency, and route risk through data modeling, DAX measures, and interactive visuals to support logistics decision-making.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Metrics
- On-Time Delivery %
- Average Transit Days
- Shipment Cost
- Route Performance
- Cost Efficiency
- Delivery Performance

## Project Highlights
- Built an interactive dashboard for freight and logistics analysis
- Used DAX measures for KPI calculation and performance tracking
- Analyzed delivery trends, transit delays, and route-level risk
- Designed visuals to support operational and business decisions
